# Urdu-English Mapping for Tasks

## Verbs (Actions)
- Add: شامل کریں (Shamil karein), ڈالو (Dalo), Add karo, Rakhdo
- List: دکھائیں (Dikhayen), فہرست (Fehrist), List dikhao, Mere kaam batao
- Delete: ختم کریں (Khatam karein), اڑائیں (Urhayen), Delete kardo, Hata do
- Complete: مکمل (Mukammal), ہو گیا (Ho gaya), Done karo, Khatam ho gaya
- Update: تبدیلی (Tabdeeli), Badlo, Change karo

## Time Phrases
- Today: آج (Aaj)
- Tomorrow: کل (Kal)
- Next week: اگلے ہفتے (Agle hafte)
- Morning: صبح (Subah)
- Evening: شام (Shaam)
- Night: رات (Raat)
