# UI/UX Components for TaskFlow

This directory contains all the UI/UX components and design documentation for the TaskFlow application.

## Components

### Layout Components
- `dashboard-layout.tsx` - Main dashboard layout with sidebar navigation
- `page-header.tsx` - Consistent page header with breadcrumbs and actions

### Task Management Components
- `enhanced-task-form.tsx` - Advanced task creation/editing form with due dates and priorities
- `enhanced-task-list.tsx` - Filterable and sortable task list with multiple view options

### UI Components
- `modal.tsx` - Reusable modal component for dialogs
- `notification.tsx` - Notification system for user feedback
- `loading-spinner.tsx` - Loading state indicators

## Design Documentation

- `UIUX-DESIGN-DECISIONS.md` - Complete documentation of UI/UX design decisions and patterns

## Key Features

### Dashboard Layout
- Responsive sidebar navigation
- Top navigation with search and user profile
- Mobile-friendly hamburger menu

### Task Management
- Create, read, update, and delete tasks
- Filter tasks by status (all, active, completed)
- Sort tasks by date, title, or priority
- Set due dates and priorities for tasks
- Edit tasks in modal dialogs

### User Experience
- Notification system for user feedback
- Loading states for better perceived performance
- Consistent design language across all components
- Responsive design for all device sizes
- Accessibility features including keyboard navigation

## Usage

These components are designed to be reusable across the application. Each component is self-contained and follows consistent design patterns.
