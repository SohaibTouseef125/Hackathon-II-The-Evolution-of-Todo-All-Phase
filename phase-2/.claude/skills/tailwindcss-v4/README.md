# Tailwind CSS v4 Skill

This skill provides comprehensive guidance for Tailwind CSS v4 development, covering utility-first styling, responsive design, dark mode, custom components, and modern styling techniques.

## Overview

The Tailwind CSS v4 skill enables Claude to work effectively with Tailwind CSS v4 for creating responsive, accessible, and maintainable web interfaces. It covers the latest features in Tailwind CSS v4, utility-first styling principles, responsive design patterns, dark mode implementation, custom component creation, performance optimization, and integration with modern frameworks like React, Vue, and Next.js.

## Structure

The skill is organized into the following components:

### SKILL.md
The main skill file containing:
- Core capabilities for Tailwind CSS v4 development
- Configuration and setup guidance
- Responsive design and breakpoint management
- Dark mode and color scheme implementation
- Custom component creation and reusability
- Performance optimization techniques
- Framework integration patterns
- Accessibility guidelines and best practices

### references/
Detailed documentation and reference materials:
- Tailwind CSS v4 API reference
- Configuration options and utilities
- Migration guide from v3 to v4
- Component patterns and recipes
- Troubleshooting reference

### assets/
Template files and boilerplate code:
- Tailwind configuration templates
- Component boilerplate files (HTML/JSX)
- Project starter templates
- Form and navigation components
- Package.json and PostCSS configurations

### scripts/
Helper scripts for Tailwind development tasks:
- Configuration validation and setup
- Component generation tools
- Project setup automation
- Version checking utilities
- Migration helpers

## Usage

This skill should be used when working on:
- New Tailwind CSS v4 projects and configurations
- Migration from Tailwind v3 to v4
- Creating responsive, accessible web interfaces
- Building design systems and component libraries
- Performance optimization for Tailwind projects
- Integration with React, Vue, Next.js, or other frameworks
- Custom theme and plugin development
- Troubleshooting complex Tailwind implementations
- Accessibility compliance for Tailwind projects
